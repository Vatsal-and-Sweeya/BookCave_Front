About

The website enables you to download books with ease!

  • Idea behind the tool

    We employ distance searching algorithms and subsequent string pattern matches to provide the best download for the title you search for.
    Unlike other websites, you don't have to go through the hassle of looking up links, you just write the title of the book and hit enter and the website downloads the book for you.

  • Extension Preference

    We have created a python tool which works in tandem with the website which can be found on GitHub feel free to go through it!
    We give preference to .pdf then .epub then .mobi then .fb2 you may change this however you like in the source code!

  • Dependencies

    We work with Python(host version 3.9.9) for functionality and use Django(host version 4.0) for web integration. The website is hosted on Heroku and is intergated with Git so it auto-updates as versions of libraries and open source contributions improve.
